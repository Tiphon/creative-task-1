# Pillow
Pillow- Билблитека в puthon, c помощью которой можно обрабатывать изображения
H2 что умеет эта библиотека:
1.  открывать изображения
2.  редактирорвать изображения
3.  обрезать картинку
4.  выводить форматы изображений(jpeg, png...)
